# Milad's Recipe

This is my website for searching in recipes, I used [**Spoonacular**](https://spoonacular.com/) API in this webpage.

# How to use

1. [Register](https://spoonacular.com/food-api/console#Plan) in Spoonacular Recipe search and get you ID and KEY
1. Clone this repository
1. Create a .env file in your project directory
    ```
     REACT_APP_ID=your ID
     REACT_APP_KEY=your Key
    ```
1. > npm install
1. > npm start
